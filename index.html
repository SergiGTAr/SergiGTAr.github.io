Projects

Projects that I did on my free time or during studies.

AES Project

Social network project developed from scratch with the MEAN Stack. The project was not fully functional, but it has the basic functionalities of a social network.

Android Racer

Small project of a videogame tribute to the Formula 1 driver Fernando Alonso "El Nano". It is developed in Kotlin using libGDX as engine and runs only on Android.

Noti-JA!

News application developed with Kotlin, using Jetpack Compose and connecting it to a news API.

The project ended up being fully functional, lacking to improve the aesthetics and add extra features.

Videogame Management

Java project using Hibernate to create a video game management system. It is an application where you can add video game companies, video games... Creating a catalog of them. The purpose of it is to make a CRUD in Java using JPA.

About

Junior software developer with one year of on-the-job experience and in second year of a higher degree training cycle in multiplatform application development.

I am currently working with SAP Business One. 
On my own and in studies I am working with Android, Java, Kotlin, Python, JavaScript, TypeScript, Angular, NodeJS, Docker among other development technologies.
